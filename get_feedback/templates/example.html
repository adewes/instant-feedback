This is an example survey. It has been created especially for you to show you different feedback elements that you can use for your website.

How it works

This tool allows you to integrate simple, intuitive feedback elements directly into your own website. You do this by simpyl pasting a small code snippet into your website's HTML and adding feedback elements in form of simple SPAN tags wherever you want to collect feedback. Like this, you can collect user feedback without interrupting the user experience.

How much do you like this?

As a bonus, since the feedback elements are realized using very basic HTML they automatically adapt to your page's CSS and integrate seamlessly into your website.

Available elements

As of today, we offer five different feedback elements that you can integrate into your website to collect user feedback. Here they are:

Feedback Action Example
Vote
Check
Rate
Write
Choose

As you've maybe noticed, as the survey admin you can click on the icon next to each element to easily modify it in place. In addition, you can get a full results summary by just clicking on the toggle results link in the admin menu above. Try it out!
