Hi there!

This tool allows you to integrate simple, intuitive feedback elements directly into your own website. You do this by simply pasting a small code snippet into your website's HTML and adding elements in form of simple SPAN tags wherever you want to collect feedback. Like this, you can collect user feedback without interrupting the user experience.

How much do you like this idea?

As a bonus, since the feedback elements are realized using very basic HTML they automatically adapt to your page's CSS and integrate seamlessly with the rest of your user experience.

Available elements

You can choose between five different feedback elements:

Feedback Action Example
Vote
Check
Rate
Write
Choose

As you've maybe noticed, if you're logged in as admin you can click on the icon next to each element and modify it in place. In addition, you can get a full results summary by just clicking on the "toggle results" link in the admin menu above. Try it out!
