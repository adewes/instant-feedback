How to use the survey

It's really easy.

1.) Copy & Paste Code

Just take the following code and put it into the HEAD section of your HTML file:

<link href="/static/feature_styles.css" rel="stylesheet">
<script src="/static/feedback.js"></script>
<script>
var survey_key = '{{survey.key}}';
</script>

2.) Make sure the initialization code runs

feedback.js will call a function called initialize_elements() when the body of your document gets loaded. So, if you overwrite the onload function of the body element, make sure to call initialize_elements() by hand.

2.) Add feedback elements

Choose from our library of elements and add them to your website by putting a corresponding <span> tag in your code. Example for a "vote" element: 

<span class="survey_vote"></span>

3.) Activate elements

To activate the feedback elements you need to visit the website in which they're integrated once while being logged in as survey admin.

4.) Edit elements on the fly

When opening the website while being logged in as survey admin you will be able to edit all elements in-place. Like that you can add descriptions, change default values or add choices.
