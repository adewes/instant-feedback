General

Vesperr is a one page HTML site template built with twitter bootstrap and has responsive layout . This template coming with single custom color : 'blue' which stored in file 'assets/color/default.css' in folder color

We've separated the custom color (blue) in a single file : default.css to make you easy to understand and customize or overwrite with your own color/style

Since Vesperr template built with bootstrap so you can go to http://twitter.github.com/bootstrap/ to learn more about twitter boostrap. Also lot of basic documentation or template overview already added on each page under 'Features' menu.

Since it's HTML CSS based you can edit all text, content with HTML editor like notepad++ or anykind else. Quite easy for people with basic HTML-CSS knowledge :)

CSS files

All CSS files stored in folder 'assets/css' and there is one folder 'color' for orange color style of the template. However, you'd just need to edit the style.css file for main style and default.css for the template main color.

All orange color as seen on the template configured in assets/color/default.css so if you want to edit other color then don't go to default.css (it's only set blue color)

list of css files used in Vesperr template

All portfolio images stored in folder 'assets/img/portfolio'. You need 2 images for each portfolio item, one portfolio thumbnail image and one big image that displayed after "zoom" icon clicked

The portfolio filter functionality handled by jQuery quicksand script : 'js/portfolio/jquery.quicksand.js' and the setting file is 'js/portfolio/setting.js'

- Parallax image background :

There are 3 area in Vesperr which use parallax image background :

- Intro

- services

- bottom

The image for parallax background stored in folder 'assets/img/bg'

See how set parallax via CSS :

/* --- on section intro --- */
section#intro{
    background: url(../img/bg/bg-1.jpg) 50% 0 no-repeat fixed;
    position:relative;
    width:100%;
    height:auto;
    margin:0;
    padding:240px 0 120px;
}

/* --- on section services --- */
section#services {
    background: url(../img/bg/bg-1.jpg) 50% 0 no-repeat fixed;
    width:100%;
    height:auto;
    margin:0;
}

/* --- on section bottom --- */
section#bottom{
    background: url(../img/bg/bg-1.jpg) 50% 0 no-repeat fixed;
    width:100%;
    height:auto;
    margin:0;
    padding:40px 0 30px;
}

 

And the javscript function for parallax image background can be found in file 'custom.js'

    // parallax background
        $('#intro').parallax("50%", 0.1);
        $('#services').parallax("50%", 0.2);
        $('#bottom').parallax("50%", 0.1);

 

Icons with Fontawesome

Selecao template uses fontawesome icons with some customization on button type (cirlced, rounded, flat) and size (32px, 48px, 64px and 128px). We've written all in icons.html

You can just copy desired icon HTML code if you need to use it on other template area.

For more detailed explanation and usage of fontawesome icons you can go to http://fortawesome.github.com/Font-Awesome/

 

56 Animations
