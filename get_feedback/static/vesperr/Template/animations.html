56 Animation effects

You put following animation into any elements such as div tag, button, image..etc. Just write the animation name as shown below as the class name and make sure you add e_ before the animation name. For example to use "flash" animation you should write .e_flash

Just hover on each button to see how animation works.

flash bounce shake tada swing wobble wiggle pulse
